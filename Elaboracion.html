Jimar

Para la elaboracion de la Raicilla Hnos. Arrizón primeramente se necesita jimar las lechuguillas que ya estan en la etapa de madurez o ya tienen floración, la jima consiste en quitar las hojas y las raices del agave de tal modo que solo quede la piña o cabeza, para esto se utilizan hachas o coas. Para que una lechuguilla llege a su etapa de madurez o floración deben pasar al menos 8 años.

Tatemar

Una vez completa la jima se pasa a tatemar o cocer las piñas del agave o tambien conocidas como "mezcal", para tatemar el mezcal primero se caliente el horno hecho con piedra y ladrillo, con madera a mas de 1000° C, hasta que el horno se vea rojo; una vez caliente el horno se le quitan los residuos del fuego con un rastrillo de madera y enseguida se introducen las piñas y despues se sella la puerta del horno con madera y lodo, este proceso dura a lo largo de 72 horas.

Triturar

Una vez tatemado el mezcal se pasa a triturar o despedazar las piñas, esto se hace en una canoa de madera y mazos tambien de madera, todo esto se hace a mano.

Fermentar

Una vez triturado el mezcal se pasa a la fermentación donde se hecha el mezcal triturado o tambien llamdo "Bagazo" a los tambos de fermentación de acero inoxidable o barriles de madera, donde se le agrega un poco de agua, este proceso tarde entre 5 a 8 dias, dependiendo el clima.

Destilar

Cuando ya esta listo el fermento se hecha al destilador, donde se le pone fuego con madera para que así salga el raicilla ya destilado.

Embotellar

Teniendo ya lista la raicilla destilada se pasa a embotellar y etiquetar, esto se hace manualmente.
